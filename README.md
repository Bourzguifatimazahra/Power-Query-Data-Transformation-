# **Power Query Data Transformation Challenge**

**Bienvenue**, une série de challenges pratiques conçus pour maîtriser la transformation de données avec **Power BI / Power Query**.  
Chaque défi présente un jeu de données réaliste et imparfait que vous devrez nettoyer, restructurer et préparer pour obtenir le résultat cible illustré.

---

## **Objectif**

L'objectif est de développer une expertise opérationnelle en :
- **Diagnostic** des problèmes de qualité des données
- **Application** de transformations avancées dans Power Query
- **Reproduction exacte** des outputs attendus
- **Construction** d'une méthodologie analytique robuste, adaptée aux environnements professionnels

Chaque challenge comprend :
-  Un jeu de données source (CSV/Excel)
-  Une visualisation du résultat attendu
-  Documentation et références associées
 
## **Stack Technologique & Compétences**

| Compétence | Description | Outils |
|------------|-------------|--------|
| **Diagnostic Data** | Identification d'anomalies, valeurs manquantes, incohérences | ![Power Query](https://img.shields.io/badge/Power_Query-2BA579?style=flat&logo=powerquery&logoColor=white) |
| **Transformation** | Nettoyage, standardisation, restructuration | ![Power BI](https://img.shields.io/badge/Power_BI-F2C811?style=flat&logo=powerbi&logoColor=black) |
| **Manipulation Avancée** | Pivot, unpivot, merges, agrégations dynamiques | ![M Language](https://img.shields.io/badge/M_Language-0066CC?style=flat&logo=microsoft&logoColor=white) |
| **Optimisation** | Gestion des types, performances, paramétrisation | ![Excel](https://img.shields.io/badge/Excel-217346?style=flat&logo=microsoftexcel&logoColor=white) |

**Compétences clés développées :**
-  Détection et correction d'erreurs
-  Standardisation des formats (dates, textes, nombres)
-  Nettoyage avancé (Trim, Clean, Replace Values)
-  Création de colonnes personnalisées
-  Agrégations et regroupements (Group By)
-  Fusion et jointures (Merge/Append Queries)
-  Préparation pour reporting et visualisation

---

**Workflow Recommandé :**
1. **Exploration** - Comprenez la structure et les problèmes
2. **Planification** - Établissez votre stratégie de transformation
3. **Exécution** - Appliquez les étapes dans Power Query
4. **Validation** - Comparez avec le résultat attendu
5. **Documentation** - Notez vos transformations clés
